This repository contains assignments, projects, and lab work for the Computer Science & Engineering (CSE) & (CS) program at **BRAC University**. 

##  Course List

### CSE 110: Programming Language I
Introduction to the foundations of computation, problem analysis, algorithm development, and programming basics.

### CSE 111: Programming Language II
Covers data structures, formal syntax, language theory, and structured programming concepts.

### CSE 220: Data Structures
Focuses on data organization, algorithms, and structures like arrays, lists, stacks, queues, and trees.

### CSE 221: Algorithms
Study of efficient algorithms, design techniques, graph algorithms, and complexity analysis.

### CSE 321: Operating Systems
Principles of operating systems, including process management, scheduling, file systems, and security.

### CSE 330: Numerical Methods
Numerical methods for root finding, linear equations, interpolation, and differentiation/integration.

### CSE 370: Database Systems
Introduction to database models, SQL, management systems, and database design principles.

### CSE 391: Programming for the Internet
Survey of Internet technologies and web programming using Client/Server structures and web tools.

### CSE 420: Compiler Design
Theory and practice of compiler design, including parsing, lexical analysis, and code generation.

### CSE 421: Computer Networks
Basics of computer networks, protocols (TCP/IP), LANs/WANs, and network security.

### CSE 422: Artificial Intelligence
Concepts in AI, including knowledge representation, search techniques, and machine learning.

### CSE 423: Computer Graphics
Fundamentals of computer graphics, 2D/3D viewing, transformations, and rendering algorithms.

### CSE 470: Software Engineering
Software engineering concepts, including requirements, design, verification, and project management.

### CSE 471: Systems Analysis and Design
Systems analysis and design using tools like data flow and E-R diagrams.

### CSE 489: App Development
Focuses on application development methodologies and practices.

### MAT 120: Mathematics
Fundamental mathematics coursework required for computer science.

### STA 201: Statistics
Coursework covering statistics and probability theory.
